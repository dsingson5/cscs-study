Day 3 — architecture comparison

Two side-by-side implementations of the same lesson (EC coupling & NMJ). Open each on your phone and laptop to feel the difference.

Full React

day3-react.html · ~15 KB source + ~280 KB libs

Built with React 18 + Framer Motion + Tailwind, all loaded from CDN (no build step). Single-page rewrite from scratch.

  • Animated neuron banner draws itself on load (SVG path animation)
  • EC coupling step-walkthrough with smooth state transitions and Auto-play
  • Underlined technical terms in body text → hover for animated popover
  • Question cards stagger-in as you scroll; reveal block animates open
  • Tailwind handles polish and consistent design tokens
  • Caveat: ~280 KB of CDN libs vs. ~100 KB total in the static page
 Hybrid

day3-hybrid.html · existing static + 12 KB React layer

The same lesson page you'd see at the dated URL, plus a small React + Framer Motion layer that adds term popovers to body text. Nothing else changes.

  • Static page generated by Python (same as today's homepage)
  • React loaded from CDN, used only for the term popover component
  • Hover any underlined term (ACh, T-tubules, sarcoplasmic reticulum, troponin, cross-bridge…) → animated definition card
  • Adds ~12 KB to the existing page; no other architecture change
  • All the existing widgets, glossary, SM-2 review queue still work
